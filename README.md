# 小程序脚手架
> 快速启动一个个人小程序

## 目录结构
```
+-- api                       存放api相关配置
+-- assets                    公共静态资源
|   +-- data                  公共常量数据
|   +-- font                  公共字体图标
|   +-- images                公共静态图片
|   +-- style                 公共样式
+-- components                小程序组件
+-- pages                     小程序页面
+-- utils                     小程序工具函数
|   +-- lodash.js             基础函数
|   +-- wxUtils.js            promise微信函数
+-- .gitignore                git忽略文件
+-- app.js                    小程序入口文件
+-- app.json                  小程序入口配置
+-- app.wxss                  小程序入口样式
+-- config.js                 全局依赖配置
+-- project.config.json       全局项目配置
+-- Readme.md
```

## 启动项目

* 1. 拉取项目
* 2. 在微信开发工具中添加项目目录

## 基础库依赖
